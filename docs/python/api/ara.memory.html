class ara.memory.knowledge.Scratchpad(text: str = 'Nothing yet!', prev_text: str = 'Nothing yet!')[source]¶

Bases: object

Ephemeral per-character notepad carried across scene boundaries.

Characters can overwrite their scratchpad via the write_scratch tool. The previous scene’s scratch is retained as prev_text so the model can reference it at the start of a new scene.
