class ara.models.GameRole(*values)[source]¶

Bases: StrEnum

Enumeration of LLM roles used within the engine.

Each role receives distinct temperature and system-prompt settings.

CHARACTER = 'character'¶

An in-world NPC or the player character.

NARRATOR = 'narrator'¶

The visual-novel narrator who describes environment and mood.

ORCHESTRATOR = 'orchestrator'¶

The director/DM that decides who speaks next and how the scene advances.

class ara.models.Importance(*values)[source]¶

Bases: IntEnum

Character importance levels controlling memory fidelity and scratch access.

Higher importance characters receive end-of-scene scratch updates and persistent conversation storage.

ANONYMOUS = 0¶

Background character with minimal persistence.

EIGEN = 3¶

Protagonist-level character with highest persistence.

IMPORTANT = 1¶

Significant character with scratch updates and memory storage.

REQUIRED = 2¶

Plot-critical character.

class ara.models.StreamResult(content: str = '', reasoning_content: str = '', tool_calls: list[dict[str, ~typing.Any]]=<factory>)[source]¶

Bases: object

Accumulator for a streamed LLM response.

Variables:

  • content – Final response text produced by the model.

  • reasoning_content – Chain-of-thought text when DeepSeek thinking mode is enabled. May be empty for non-thinking models.

  • tool_calls – Parsed tool-call descriptors accumulated from delta chunks.
