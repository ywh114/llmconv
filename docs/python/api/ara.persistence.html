ara.persistence package¶

Persistence layer for Ara.

class ara.persistence.SaveManager(settings: AraSettings)[source]¶

Bases: object

Manages save files for Ara stories.

Saves are stored as JSON files under <settings.saves_path>/<story_id>/slot_<N>.json. Each save captures a full snapshot of story + engine + character state.

ara.persistence.save module¶

VN-style save/load system for Ara stories.

Saves capture the full runtime state of a Story so that play can be resumed later from the exact same point.
