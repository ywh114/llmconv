class ara.webclient.proxy.DirectProxy(server: AgentServer)[source]¶

Bases: object

Direct in-process proxy to an AgentServer instance.

Avoids UNIX socket overhead by calling AgentServer.handle_request() directly. Thread-safe via the server’s internal lock.

ara.webclient.server module¶

Starlette web gateway for the Ara VN frontend.

Simple JSON proxy to the agent server - no SSE, no streaming. The frontend polls /next to advance the story and /input to reply.
