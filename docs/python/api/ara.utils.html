ara.utils.debug module¶

Interactive debug console for the Ara engine.

Inspired by the old project’s get_user_debug hook, the console provides real-time introspection into the conversation state, LLM context, and engine internals. It can be triggered automatically every turn or on-demand via /command or :command at the player prompt.
