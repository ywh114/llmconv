Bases: BaseSettings

Single consolidated settings class for the Ara engine.

Values are read from the [tool.ara] table in pyproject.toml and may be overridden by environment variables prefixed with ARA_.

Variables:

  • api_key – API key for the LLM provider. Falls back to the DEEPSEEK_API_KEY environment variable if empty.

  • api_endpoint – Base URL for the LLM API.

  • api_model – Model identifier sent to the provider.

  • embedding_model – sentence-transformers model name used by ChromaDB.

  • data_dir – Root directory for persistent data (ChromaDB, assets, etc.).

  • language – Default language for generated text.

  • temperature_character – Sampling temperature for character turns.

  • temperature_narrator – Sampling temperature for narrator turns.

  • temperature_orchestrator – Sampling temperature for orchestrator turns.

  • strict_tools – When True, request the DeepSeek beta endpoint and emit strict: true in tool schemas for guaranteed JSON schema compliance.
